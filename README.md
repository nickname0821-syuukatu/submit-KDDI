## 成果物ファイル

### 概要

大学における研究活動の一環として作成した、As-Rigid-As-Possible（ARAP）Shape Modeling に基づく3次元メッシュ変形プログラムです。
入力された3Dメッシュに対して拘束条件を与え、局所的な形状特徴を保持しながら自然な変形を行います。

## ファイル構成

### ARAP_addition.cpp

ARAPアルゴリズムを実装したソースコードです。

主な処理

* OBJファイルの読み込み
* メッシュの局所形状特徴（LSRV）から関節候補を自動推定
* 関節領域と非関節領域で異なる回転平滑化を適用
* ARAPベースのメッシュ変形を実行

### horse_new.obj

### パソコン室.wrl

講義課題として作成したVRMLファイルです。
研究室内の机、椅子、PC、棚などを配置し、パソコン室の内装を3次元空間上で再現しました。
VRMLによるシーン構築および3次元モデリングの学習を目的として作成しました。

### 東京（一部）再現.wrl
講義課題として作成したVRMLファイルです。
東京のある場所での高さを1m感覚で記録されたファイルから、pythonを使ってポリゴンをつくり、地面と建物の判定をしました。
なお、ファイルサイズが大きいため表示には時間がかかります。
